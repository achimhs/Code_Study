2. **하위 컴포넌트** `SubjectComp`

![](../img/8_19_4.png)

script 문안에 vue.component를 이용해서 이름이 `SubjectComp`라는 컴포넌트를 구현하여라.

- data : `count`- int형
- prop : `title`- String형, 필수값
- 버튼 : 버튼을 누르면 `addCount`라는 메서드를 호출한다.

- data : `total`- int형
- 메서드 : `addTotalCount`

  - `total` 1 증가

  - `total`이 10이라면 "**투표 종료~**"라는 문구가 담긴 알림창이 뜬다.

![](../img/8_19_6.png)
